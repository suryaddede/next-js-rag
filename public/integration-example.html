Example HTML Page

This is an example of how to integrate the RAG Chat Widget into your HTML webpage.

Integration Methods:

  1. JavaScript Widget (Recommended): Add the script tag below to your HTML page
  2. iframe Method: Use direct iframe embedding for simpler integration

Method 1: JavaScript Widget Integration

Add this script tag to your HTML page:

<script src="http://localhost:3000/chat-widget.js"></script>

For custom configuration:

<script>
window.RAGChatWidget.init({
    baseUrl: 'http://your-domain.com',
    position: 'bottom-right',
    buttonColor: '#059669',
    title: 'Support Chat',
    maxWidth: '400px',
    maxHeight: '600px'
});
</script>

Method 2: Direct iframe Integration

For simpler integration, you can use a direct iframe:

<iframe 
    src="http://localhost:3000/embed" 
    style="position: fixed; bottom: 0; right: 0; width: 100vw; height: 100vh; border: none; pointer-events: none; z-index: 1000;"
    onload="this.style.pointerEvents='auto'">
</iframe>

HTML Integration Example

<!DOCTYPE html>
<html>
<head>
    <title>My Website</title>
</head>
<body>
    <h1>Welcome to our website</h1>
    <p>Your content here...</p>

    <!-- Add the chat widget -->
    <script src="http://localhost:3000/chat-widget.js"></script>
</body>
</html>

The chat widget will appear as a floating button in the bottom-right corner of your page. Users can click it to open the chat interface.
